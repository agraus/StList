Учебный проект "Список студентов".
Возможности:
Добавление в базу студентов в бд с валидацией данных и проверкой уникальности почты.
Вывод таблицы с ограничением в 4 студента на страницу с постраничной навигацией.
Поиск по таблице.

Используется: PHP 7, MySQL, Composer.
При установке необходимо добавить в /data/settings.xml настройки для своей базы.
